# FlutterPlugins
flutter的插件集合


## device_info_csx
补充pub.dev上面的device_info内容，可以完全替换该仓库


## background_alive_csx
支持后台一段时间保活的操作


## limiting_direction_csx
目前唯一一个有效的支持iOS端控制设备支持方向的对应旋转插件


## image_compress_csx
图片压缩插件


## ios_specific_csx
iOS 独有的一些操作，比如iOS的健康管理


## common_plugin_csx
一些小组件插件：eg：文件的md5


## doraemonkit_csx
 didi的doraemonkit转换flutter使用demo
